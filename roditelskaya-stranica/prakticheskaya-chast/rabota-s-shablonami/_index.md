Шаблоны используются для оформления типового текста: например, описания функций системы, технического задания, тест-кейсов.

-  Шаблон создается для каталога и сохраняется в репозитории.

-  Шаблон можно применить в любой статье.

## Создание шаблона

В меню каталога *Инструменты* -> *Шаблоны* -> *\+ Новый шаблон*.
